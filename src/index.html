My Web Components

This is a collection of predefined, accessible web components I've created, that are available for use in different projects

The main focus for the components, are that they shall follow WCAG-standards for accessibility, and be predefined with WAI-ARIA labels where needed. For this project, I'm also aiming for a few different styles, that the components can be sorted by. For the styles, I'm looking for "glassy"-look, skeumorphism,
